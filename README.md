奈倉柏木判定プログラム
====

wjnの「[実録]女のSEX告白」記事の書き手を文章から判定するプログラムです。

## Requirement
- python3
- [pip_requirements](https://github.com/hanarchy/DetectWjnAuthor/pip_requirements)

## Usage
### 文章を直接入れる
`python3 detect_author.py -s [判定したい文章]`

### wjnの記事URLを入れる
`python3 detect_author.py -o [url]`
